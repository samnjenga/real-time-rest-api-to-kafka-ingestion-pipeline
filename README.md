# Real-Time REST API to Kafka Ingestion Pipeline

A real-time ingestion pipeline that streams data from REST APIs into Kafka for low-latency processing and analytics.

---

## Overview

This project demonstrates a scalable ingestion pipeline that continuously fetches data from a REST API and publishes it to Kafka.

Example: Streams BTC price data from Binance API into Kafka in near real-time.

---

## Architecture

REST API → Ingestion Service → Kafka → Consumers

---

## Key Features

- Continuous API ingestion
- Kafka producer integration
- Configurable polling interval
- JSON data processing
- Scalable design

---

## Use Cases

- Real-time analytics
- Market data ingestion
- Event-driven systems

---

## Tech Stack

- Apache Kafka
